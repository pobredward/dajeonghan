# 00. 다정한(dajeonghan) - 프로젝트 개요 v2

> **"공간 중심의 생활 OS - 당신의 집이 스스로 말합니다"**

## 📱 프로젝트 비전

### 다정한이란?

**다정한**은 집의 모든 공간과 물건을 시각화하여 관리하는 차세대 생활 비서 앱입니다. 
단순한 할 일 목록이 아닌, **실제 공간과 1:1 매핑되는 디지털 트윈**을 만들어 직관적이고 효과적인 생활 관리를 제공합니다.

### 핵심 차별화 (v2 업데이트)

#### 🏠 공간 기반 아키텍처
```
기존 앱: 카테고리별 리스트 → 추상적, 기억 부담
다정한: 실제 집 구조 → 직관적, 현실 매핑

[예시]
기존: "청소" 메뉴 → 15개 할 일 나열
다정한: 내 집 지도 → 주방(빨간색) → 냉장고 클릭 → 3개 할 일
```

#### 🎨 시각적 상태 피드백
```
공간과 아이템이 "상태"를 가짐
- 깨끗함: ✨ 90-100점, 밝고 선명
- 보통: 😊 70-89점, 일반
- 주의: ⏰ 40-69점, 약간 흐림
- 긴급: ⚠️ 0-39점, 흐리고 알림

사용자는 "점수"로 게임처럼 관리
부정적 피드백(더러움)은 옵션
```

#### 🔧 무한 확장 시스템
```
초기: 주방(냉장고, 싱크대) + 화장실 + 침실
→ 거실(소파, TV, 공기청정기)
→ 베란다(화분, 세탁물)
→ 차량(정기점검, 세차)
→ 반려동물(사료, 산책, 미용)
→ 사용자 커스텀 무한 추가
```

## 🏗️ v2 아키텍처 개요

### 계층 구조

```
┌─────────────────────────────────────────────────────┐
│           다정한 앱 (v2 - 공간 기반)                  │
├─────────────────────────────────────────────────────┤
│  Visualization Layer (시각화)                       │
│  ├── 집 지도 뷰 (Home Map)                          │
│  ├── 공간 상태 표시 (Room Status)                   │
│  ├── 아이템 3D/2D 시각화                             │
│  └── 점수/게이지 시스템                              │
├─────────────────────────────────────────────────────┤
│  Space Management Layer (공간 관리)                 │
│  ├── HomeLayout (집 구조)                           │
│  ├── Room (방)                                      │
│  ├── Item (아이템: 냉장고, 세탁기 등)                │
│  └── Feature (기능: 청소, 재고, 유지보수)            │
├─────────────────────────────────────────────────────┤
│  Core Engine Layer (핵심 엔진)                      │
│  ├── RecurrenceEngine (주기 계산)                   │
│  ├── PriorityCalculator (우선순위)                  │
│  ├── VisualFeedbackEngine (시각 피드백) 🆕          │
│  └── ScoreCalculator (점수 계산) 🆕                 │
├─────────────────────────────────────────────────────┤
│  Data Layer (데이터)                                │
│  ├── Firestore (클라우드)                           │
│  ├── Local Storage (오프라인)                       │
│  └── Item Presets (아이템 프리셋 DB)                │
└─────────────────────────────────────────────────────┘
```

### 데이터 흐름

```
1. 온보딩: 집 구조 설정
   사용자 → 집 유형 선택 → 방 배치 → 아이템 배치
   
2. 일상 사용: 공간 중심 탐색
   홈 화면 → 집 지도 보기 → 주방(점수 45) 클릭
   → 냉장고(빨간색) → 3개 할 일 보기 → 완료
   
3. 자동 업데이트: 상태 반영
   할 일 완료 → 점수 +10 → 냉장고 밝아짐
   → 주방 점수 55로 상승 → 전체 집 점수 상승
```

## 🎯 MVP 범위 (v2)

### 포함 기능

#### 1. 공간 시스템 (핵심 🆕)
- ✅ 집 유형 선택 (원룸, 투룸, 아파트, 커스텀)
- ✅ Drag & Drop 방 배치
- ✅ 아이템 팔레트 (필수/선택)
- ✅ 공간별 점수 계산
- ✅ 3가지 뷰 모드 (오늘/지도/리스트)

#### 2. 시각화 시스템 (핵심 🆕)
- ✅ 점수 기반 색상/필터
- ✅ 상태별 애니메이션
- ✅ 프로그레스 게이지
- ✅ 배지 및 성취 시스템
- ⚠️ 시각적 더러움 (옵션, 기본 OFF)

#### 3. 기본 공간 및 아이템
**주방**
- ✅ 냉장고 (청소 + 재고)
- ✅ 싱크대 (청소 + 생필품)
- ✅ 가스레인지 (청소)

**화장실**
- ✅ 변기/세면대 (청소)
- ✅ 세탁기 (청소 + 유지보수)

**침실**
- ✅ 침대/침구 (청소)
- ✅ 약 보관함 (재고 + 알림)

**거실**
- ✅ 소파 (청소)
- ✅ TV/전자기기 (먼지 제거)

#### 4. 기존 기능 유지
- ✅ 공통 엔진 (주기, 우선순위, 미루기)
- ✅ 식재료 데이터베이스 (300+)
- ✅ 알림 시스템 (다이제스트)
- ✅ 오프라인 작동

### 제외 기능 (Phase 2+)

- ⏳ 베란다, 현관 공간
- ⏳ 차량 관리
- ⏳ 반려동물 관리
- ⏳ 3D 시각화
- ⏳ AR 가구 배치
- ⏳ 커뮤니티 프리셋 공유

## 🎨 UX 플로우 (v2)

### 첫 사용 경험

```
1. 스플래시 (1초)
   → 로고 + "다정한" 애니메이션

2. 온보딩 Step 1: 집 선택 (15초)
   "어떤 집에 사시나요?"
   [원룸] [투룸] [아파트] [직접 구성]

3. 온보딩 Step 2: 방 배치 (30초)
   선택한 유형의 기본 레이아웃 표시
   "방을 드래그하여 배치하세요"
   (건너뛰기 가능)

4. 온보딩 Step 3: 아이템 선택 (각 방마다 30초)
   "주방에 있는 것을 선택하세요"
   [냉장고✓] [싱크대✓] [전자레인지] [식기세척기]
   필수 아이템은 미리 체크

5. 온보딩 Step 4: 첫 할 일 생성 (5초)
   "분석 중..." → 프로그레스 바
   → "오늘 할 일 5개 준비 완료!" 🎉

6. 홈 화면 (메인)
   → 집 지도 보기
   → 오늘 할 일 보기
   → 10분 코스 시작
```

### 일상 사용 패턴

```
아침 (09:00)
→ 알림: "☀️ 오늘의 할 일 5개"
→ 앱 열기
→ [지도 뷰] 주방(45점, 주황색) 확인
→ 주방 탭 → 냉장고 클릭
→ "냉장고 선반 청소 (5분)" 완료 버튼
→ 점수 45→55로 상승, 색상 노란색으로
→ 축하 애니메이션 + "10점 올랐어요!" 🎉

저녁 (20:00)
→ 알림: "🌙 오늘 남은 일 2개"
→ 빠른 확인
→ 하나 완료, 하나 미루기 (내일로)
```

## 🔬 핵심 혁신 요소

### 1. 공간의 디지털 트윈
```
현실 세계          디지털 세계
내 집 주방    →    앱 속 주방
냉장고       →    디지털 냉장고 (상태 추적)
양파(실물)   →    데이터 (D-3 알림)
```

### 2. 게이미피케이션
```
RPG 게임처럼
- HP: 공간 점수 (0-100)
- 레벨: 연속 관리 일수
- 퀘스트: 오늘의 할 일
- 보상: 배지, 칭찬, 통계
```

### 3. 점진적 복잡도
```
Day 1: 주방 냉장고만
Day 7: 화장실 추가
Day 14: 거실 추가
```

## 💡 왜 이 구조인가?

### 기존 문제점
```
❌ 카테고리별 리스트
   → "청소" 15개 → 압도감
   → 어디부터? 무엇부터?
   
❌ 추상적 할 일
   → "주방 청소" → 주방 어디?
   → 기억 부담 높음
```

### v2 해결책
```
✅ 공간 기반 탐색
   → 시각적으로 보임
   → 클릭하면 바로 그 공간 할 일
   
✅ 구체적 액션
   → "냉장고 선반 청소"
   → 정확히 무엇을 해야 하는지 명확
```
